# AUREX — Centro de Analisis y Documentacion

## Archivos en esta carpeta

| Archivo | Descripcion |
|---------|-------------|
| `ROADMAP.md` | Plan completo del proyecto — fases, hitos, estimaciones |
| `OBJETIVOS.md` | Metas mensuales, compounding proyectado, indicadores de salud |
| `REGISTRO_CUENTA.md` | Historial de todos los trades, metricas, lecciones aprendidas |

## Como usar esto

- **Al inicio de cada semana:** Revisar OBJETIVOS.md — estamos on track?
- **Al cierre de cada mes:** Actualizar REGISTRO_CUENTA.md con metricas reales
- **Cuando hay un trade importante:** Añadir leccion aprendida a REGISTRO_CUENTA.md
- **Cuando se activa una nueva estrategia:** Actualizar ROADMAP.md
